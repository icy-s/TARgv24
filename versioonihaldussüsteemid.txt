Как сделать так, чтобы при работе с GitHub не заменять файлы в оригинальной ветке (main)?

Чтобы не заменять файлы в оригинальной ветке (main), работайте в отдельной ветке. Вот как это сделать:

Создайте новую ветку:

bash
Kopeeri kood
git checkout -b имя_ветки
Работайте в этой ветке:
Вносите изменения и коммитьте их.

Отправьте изменения:

bash
Kopeeri kood
git push origin имя_ветки
Создайте Pull Request:
На GitHub предложите слияние своей ветки с main.

Так оригинальная ветка (main) останется нетронутой, пока изменения не будут проверены и одобрены.
